# Stock AI Pro Analyzer 🚀

A Spring Boot based stock analysis and binary trade simulator project.

## Features
- Live stock data using Alpha Vantage API
- Moving Average (MA20, MA50)
- RSI Indicator
- AI Buy/Sell Signal
- Confidence Score Engine
- Binary Trade Simulator UI

## Tech Stack
- Java
- Spring Boot
- REST API
- Chart.js
- Git & GitHub
